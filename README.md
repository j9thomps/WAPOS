# WAPOS Website — Version 1.0

Luxury corporate website for WAPOS, including WAPOS Club & Sauna, Male69, RENXO, and WAPOS Distribution.

## Structure

- `index.html` — homepage with first-visit intro animation
- `about.html` — WAPOS brand house
- `brands.html` — brand portfolio
- `club.html` — WAPOS Club & Sauna
- `distribution.html` — premium distribution hero with black Sprinter-style van
- `contact.html` — contact page
- `assets/css/styles.css` — global styling
- `assets/js/app.js` — language selector + intro animation
- `languages/` — Spanish default plus English, Portuguese, French, German, Italian, and Chinese

## Languages

Default language is Spanish. Included language files:

- `es.json`
- `en.json`
- `pt.json`
- `fr.json`
- `de.json`
- `it.json`
- `zh.json`

## GitHub Pages deployment

1. Upload all files in this folder to your repository:
   `https://github.com/j9thomps/WAPOS`
2. Go to GitHub → Settings → Pages.
3. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
4. Save.
5. Your GitHub Pages URL should appear after deployment.

## Domain setup

When ready, point `wapos.net` to GitHub Pages or connect it through Wix/domain DNS.

## Notes

Design direction is inspired by luxury maison/editorial websites: refined spacing, dark palette, gold accents, premium typography, and restrained motion. It does not copy Cartier assets, layout, or proprietary design.
